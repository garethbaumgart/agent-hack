# Label View

## US-18: View All Items for a Label

**As an** engineering manager
**I want to** click a label and see everything related to it
**So that** I can prepare for 1:1s or review project status

### Acceptance Criteria

- [ ] Clicking any label opens the Label View
- [ ] Shows all notes with that label (newest first)
- [ ] Shows all tasks with that label (grouped by status, sorted by due date)
- [ ] User can click a note to open it
- [ ] User can interact with tasks (mark done, edit, etc.)

---

## US-19: Navigate Back from Label View

**As an** engineering manager
**I want to** return to my previous view after reviewing a label
**So that** I can continue my workflow

### Acceptance Criteria

- [ ] User can close Label View and return to previous view (Board or Note List)
